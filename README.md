# Student Management System (SMS)

A simple web-based Student Management System built using PHP, MySQL, HTML, CSS, and JavaScript.

## 📋 Features

- Admin login
- Add/edit/delete students
- View student records
- Attendance tracking
- Report generation

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP (Apache Server)
- VS Code
